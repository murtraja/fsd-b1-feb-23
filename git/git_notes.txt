How to clone the repo?

1. Go to the repo link:
https://github.com/murtraja/fsd-b1-feb-23

2. Click on the green code drop down
Copy the link for the HTTPS section
https://github.com/murtraja/fsd-b1-feb-23.git

3. Go to terminal / git bash
navigate to the folder in which you want to clone

4. git clone https://github.com/murtraja/fsd-b1-feb-23.git

=========================
How to update the repo?

1. go to the directory in git bash

2. git pull




Collaboration notes:
git clone

git switch master

git add

git push

So this will update master branch itself without creating pull request

PR approach
========================

git add

Choose the base branch as master
