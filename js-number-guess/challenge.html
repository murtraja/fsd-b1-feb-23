JS Challenges

Idea

To develop a number guessing game and iterate over it - adding more concepts to it as and when covered in the class

Game rules

  • Computer generates a random number (> 0, < k) that it doesn't share with you
  • The player guesses the random number n times
  • The player wins the game if it guesses it correctly

Challenge 1

Hardcode the random number in the Javascript code. Take the number range from 1 to 15
